# Crucible Forge — Next.js Homepage

## Run locally
1. Install Node.js 20 or newer.
2. Open a terminal in this folder.
3. Run `npm install`
4. Run `npm run dev`
5. Open http://localhost:3000

## Deploy on Vercel
1. Create a free GitHub repository.
2. Upload all files from this folder.
3. Sign in to Vercel and import the repository.
4. Keep the default Next.js settings and deploy.
5. In Vercel, open Settings → Domains.
6. Add `crucibleforge.org`.
7. Copy the DNS records Vercel gives you into your domain registrar.
